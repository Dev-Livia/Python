## Aula 6 - Tipos Primitivos
int (inteiro): Representa números inteiros, positivos ou negativos, sem casas decimais.

float (ponto flutuante): Utilizado para representar números reais com casas decimais.

str (string): Representa texto e pode ser criado usando aspas simples (' ') ou aspas duplas (" ").

bool (booleano): Representa valores booleanos, True (verdadeiro) ou False (falso).

NoneType (None): Representa a ausência de valor ou a falta de um valor.

complex (complexo): Utilizado para representar números complexos.
