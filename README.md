## Aula 6 - Tipos Primitivos
int (inteiro): Representa números inteiros, positivos ou negativos, sem casas decimais.

float (ponto flutuante): Utilizado para representar números reais com casas decimais.

str (string): Representa texto e pode ser criado usando aspas simples (' ') ou aspas duplas (" ").

bool (booleano): Representa valores booleanos, True (verdadeiro) ou False (falso).

NoneType (None): Representa a ausência de valor ou a falta de um valor.

complex (complexo): Utilizado para representar números complexos.

## Aula 7 - Operadores Aritméticos 

Neste documento, vamos aprender sobre os operadores aritméticos do Python e sua ordem de precedência dentro de expressões matemáticas. Vamos ver como funcionam os operadores de adição, subtração, multiplicação, divisão, exponenciação e quociente na linguagem Python.

### Operadores Aritméticos

- **Adição (+)**: Usado para somar dois valores.
  a = 5
  b = 3
  soma = a + b
**Subtração (-)**: Usado para subtrair um valor de outro.
  a = 8
  b = 3
  subtracao = a - b
**Multiplicação (*)**: Usado para multiplicar dois valores.
  a = 4
  b = 6
  multiplicacao = a * b
**Divisão (/)**: Usado para dividir um valor pelo outro.
  a = 10
  b = 2
  divisao = a / b
  **Exponenciação ( ou pow())**: Usado para elevar um número à potência de outro.
  a = 2
  b = 3
  exponenciacao = a ** b
**Quociente da Divisão (//)**: Usado para retornar a parte inteira do resultado da divisão.
  a = 10
  b = 3
  quociente = a // b
 Ordem de Precedência
### A ordem de precedência dos operadores aritméticos em Python segue a mesma ordem matemática padrão:

    Parênteses têm a maior precedência.
    Exponenciação.
    Multiplicação e Divisão (mesma precedência).
    Adição e Subtração (mesma precedência).
    Você pode usar parênteses para controlar explicitamente a ordem das operações, se necessário.
    
    Por exemplo, a expressão 2 + 3 * 4 resulta em 14, mas (2 + 3) * 4 resulta em 20.

Lembre-se de que entender a ordem de precedência dos operadores é fundamental para escrever expressões matemáticas corretas em Python e em muitas outras linguagens de programação.
